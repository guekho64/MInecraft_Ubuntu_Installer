Aquí sus mejores funciones:

  • Instala Minecraft sin demasiado esfuerzo
  • Es compatible con cualquier distro de linux que derive de Debian (En desarrollo)
  • Tiene una bonita apariencia, para que sea lo más amigable al usuario posible
  • Soporta la Instalación de el Launcher Premium de Minecraft
  • Soporta la Instalación de cualquier Launcher Offline de Minecraft
  • Capaz de instalar determinados paquetes especiales que puedan llegar a ser necesarios
  • Soporta el uso de "Aria2" como gestor de descargas, para gozar de mayor velocidad
  • Soporta de uso de "apt-fast" como gestor de paquetes, que a su vez usa "Aria2" para descargar paquetes de forma rápida
  • Totalmente en Español
  • Deja un elegante icono en su escritorio al finalizar, además de añadir uno también a su menú
  • Si así lo elige usted, es capaz de arreglar la apariencia del Launcher de Minecraft, pues algunas veces no es uniforme
  • Es un programa de Código Abierto ( Open Source )
  • ¡Gratis!
